# Kerehama Admin App

Mobile admin app for labs.kerehama.nz — manage client chats from your phone.

## Setup

```bash
cd app
npm install
npx expo start
```

Scan the QR code with **Expo Go** (iOS/Android).

## Features
- 🔐 Password login (same as web admin)
- 💬 Real-time chat list
- 💬 Reply to clients as Kerehama
- ⚙️ Settings & logout
- 🎨 Matching dark theme
